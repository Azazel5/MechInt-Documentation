## LinkedIn content

This directory is for **finalized, shareable artifacts** derived from work in `topics/` and `circuit_discovery/`.

- **What goes here**: short posts, long-form writeups, link collections (papers / videos), visuals, scripts, and drafts that are meant to be published.

- **Naming**: keep paths mirroring the research source (e.g. `topics/mech_int/arena/...`) so it’s always clear what the content is derived from.

- **Rule**: every piece should link back to the originating `notes.md` or experiment folder.
